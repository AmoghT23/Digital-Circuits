# Digital-Circuits
Implementation of Digital Circuits 

* Arithmetic - Contains all the adders, subtractors, multiplication circuit design and testbenches. 
* Code Converter - Contains all the code converters circuit design and testbenches. 
* Combinational Circuits - Contains all the combinational circuit design and testbenches. 
* Sequential Circuits - Contains all the sequential circuit design and testbenches. 
* Universal Gates - Contains all the different gates using NAND and NOR gates.
